# CapDAO 🚀

> **Decentralized Crowdfunding with Strict Funding Caps, DAO Voting & Interest Rewards**

CapDAO is a complete Web3 crowdfunding platform where every campaign has an on-chain–enforced funding cap, contributors vote on fund usage via DAO governance, and contributors earn 5% APR simple interest on their contributions.

## 🚀 Quick Start (Local Demo)

### Prerequisites
- Node.js ≥ 18
- npm ≥ 9
- MetaMask browser extension
- MongoDB Atlas account (free tier) OR local MongoDB

---

### Step 1 — Start the Hardhat Local Blockchain

```bash
cd capfund/contracts
npm install
npx hardhat node
```

> Keep this terminal running. It prints 20 test accounts with 10,000 ETH each.  
> Copy Account #0's **private key** for MetaMask import.

---

### Step 2 — Deploy the Smart Contract

Open a **new terminal**:

```bash
cd capfund/contracts
npx hardhat run scripts/deploy.js --network localhost
```

The script automatically:
- Deploys `CapFund.sol`
- Writes the contract address to `frontend/src/utils/deployedAddress.js`
- Syncs the ABI to `frontend/src/utils/abi.js`

Expected output:
```
✅ CapDAO deployed at: 0x5FbDB...
✅ ABI synced to frontend/src/utils/abi.js
✅ Contract address synced to frontend/src/utils/deployedAddress.js
```

---

### Step 3 — Configure MetaMask

1. Open MetaMask → **Add Network**
2. Network settings:
   - Name: `Hardhat Local`
   - RPC URL: `http://127.0.0.1:8545`
   - Chain ID: `31337`
   - Currency: `ETH`
3. Import Account #0 from Step 1 using its private key
4. You'll have 10,000 test ETH to demo with

---

### Step 4 — Start the Backend

```bash
cd capfund/backend
npm install
```

Create `.env`:
```env
MONGODB_URI=mongodb+srv://<user>:<password>@cluster.mongodb.net/capdao
PORT=5000
FRONTEND_URL=http://localhost:5173
```

Then:
```bash
npm start
```

Backend runs at: `http://localhost:5000`

---

### Step 5 — Start the Frontend

```bash
cd capfund/frontend
npm install
npm run dev
```

Frontend runs at: `http://localhost:5173`

## 🔐 Smart Contract (`CapFund.sol`)

### Key Functions

| Function | Access | Description |
|----------|--------|-------------|
| `createCampaign(cap, deadline)` | Anyone | Creates an on-chain campaign |
| `contribute(id)` | Anyone | Send ETH; excess auto-refunded |
| `fundInterestPool(id)` | Anyone | Top up the interest reward pool |
| `withdraw(id)` | Owner | Withdraw funds (if funded + DAO approved) |
| `claimRefund(id)` | Contributor | Refund principal + interest on failure |
| `claimInterest(id)` | Contributor | Claim accrued 5% APR interest |
| `createProposal(id, desc, type, duration, newDeadline)` | Owner | Create DAO proposal |
| `vote(id, proposalId, support)` | Contributor | Weighted Y/N vote |
| `executeProposal(id, proposalId)` | Anyone | Execute after voting window |

### Proposal Types

| Type | Value | Effect |
|------|-------|--------|
| `RELEASE_FUNDS` | 0 | Allows owner to withdraw before deadline |
| `CHANGE_DEADLINE` | 1 | Extends the campaign funding deadline |
| `CANCEL` | 2 | Cancels campaign, enables full refunds |

### Events Emitted

- `CampaignCreated(id, owner, fundingCap, deadline)`
- `ContributionMade(id, contributor, amount)`
- `ProposalCreated(campaignId, proposalId, description, pType, votingDeadline)`
- `VoteCast(campaignId, proposalId, voter, support, votingPower)`
- `ProposalExecuted(campaignId, proposalId, passed)`
- `FundsWithdrawn(id, owner, amount)`
- `RefundIssued(id, contributor, amount)`
- `InterestClaimed(campaignId, contributor, amount)`
- `InterestPoolFunded(campaignId, funder, amount)`

## 💸 Interest Calculation

CapDAO implements **simple interest** at a fixed 5% APR:

```
interest = contribution × RATE × elapsed_seconds
           ────────────────────────────────────────
                       BPS × YEAR

Where:
  RATE = 500        (5% in basis points)
  BPS  = 10,000     (basis point denominator)
  YEAR = 31,536,000 (seconds in a 365-day year)
```

**Example:** Contribute 1 ETH, hold for 30 days:
```
interest = 1 ETH × 500 × 2,592,000 / (10,000 × 31,536,000)
         = 1 ETH × 0.04109...
         ≈ 0.004109 ETH ≈ 4.1 mETH
```

> The interest pool is funded separately by the campaign owner (or anyone).  
> If the pool is empty, claiming is blocked until it's topped up.

## 🗳️ DAO Voting System

- **Voting power = ETH contributed** (1 wei = 1 vote weight)
- Each address gets **one vote per proposal** (no double-voting)
- Voting window is set by the proposal creator (1 hour minimum)
- Proposal passes if `yesVotes > noVotes` after the voting period
- **Anyone** can call `executeProposal()` after the voting window closes

## 🏆 Hackathon Demo Script

1. **Open** `http://localhost:5173` → Connect MetaMask (Hardhat account)
2. **Create** a campaign: 1 ETH goal, 30 day deadline
3. **Fund** interest pool: 0.1 ETH
4. **Contribute** as another account: 0.5 ETH
5. **Show** the live interest ticker ticking up in the Interest tab
6. **Create** a RELEASE_FUNDS proposal as owner
7. **Vote YES** as the contributor (voting power = 0.5 ETH)
8. **Execute** the proposal after the voting window
9. **Withdraw** funds as owner (DAO approved early withdrawal)
10. **Claim** interest as the contributor 💸
